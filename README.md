# WebRTC

## Requirements

* iOS 10.0 or higher

## Installation

WebRTC is available through [CocoaPods](http://cocoapods.org). To install
it, simply add the following line to your Podfile:

```ruby
pod 'WebRTC', :git => 'https://github.com/monadical/webrtc-ios-framework.git'
```
